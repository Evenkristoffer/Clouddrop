# Filopplastings-App

Dette prosjektet er en enkel nettside for å laste opp, vise og håndtere filer. Frontend bygges i HTML/CSS/JS, (eventuelt tsx hvis jeg får tid, mest sannsynlig ikke) backend i Node.js/Express, og metadata lagres i enten MariaDB eller MongoDB.

Funksjoner:
Enkel Login
Opplasting av én eller flere filer
Vise liste over filer
Last ned fil + del med link
Slett fil / slett alt
Enkel validering av filtyper/størrelse

![alt text](https://github.com/Evenkristoffer/Prosjekt_idk/blob/main/media/Untitled.png?raw=true)


I denne oppgaven viser jeg drift ved å sette opp server, database og filhåndtering. Jeg viser utvikling ved å bygge API-endepunkter, kode og lage et fungerende "grensesnitt" eller app. Jeg viser brukerstøtte ved å lage en enkel, brukervennlig løsning med popup Warnings / feilmeldinger og dokumentasjon som viser alt av problemer og nødvendige detaljer som f.eks passord til database.
